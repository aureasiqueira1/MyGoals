- Typescript

- Tailwindcss
- SQLite

- Adicionar meta

- Adicionar depósito


## ⚙️ Como executar

Clone o projeto em sua máquina
```bash
  git@github.com:aureasiqueira1/MyGoals.git
```

Abra o projeto
```bash
  cd MyGoals
```

Instale as dependencias (com yarn ou npm)
```bash
  npm install
```

Inicie o projeto (com yarn ou npm)
```bash
  npx expo start
```
