# MyGoals

App de metas financeiras pessoal

## 🎯 Tecnologias utilizadas

- React Native
- Typescript
- Expo
- Expo Router
- NativeWind
- Tailwindcss
- SQLite



## ⚙️ Como executar

Clone o projeto em sua máquina
```bash
  git@github.com:aureasiqueira1/MyGoals.git
```

Abra o projeto
```bash
  cd MyGoals
```

Instale as dependencias (com yarn ou npm)
```bash
  npm install
```

Inicie o projeto (com yarn ou npm)
```bash
  npx expo start
```
